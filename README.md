# Testes Estatísticos – TCC

Este repositório contém o notebook e os arquivos utilizados na análise estatística desenvolvida para o Trabalho de Conclusão de Curso (TCC).

## Conteúdo

- Notebook em Google Colab
- Base de dados utilizada nos testes
- Código-fonte das análises
- Resultados obtidos

## Como utilizar

1. Clone este repositório;
2. Abra o notebook no Google Colab ou Jupyter Notebook;
3. Execute as células na ordem em que estão organizadas.
